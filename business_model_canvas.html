Business Model Canvas

Système de Pointage
par Reconnaissance Faciale

Stack : React · Firebase · face-api.js · Telegram API  |  Avril 2026

🤝

Partenaires Clés

  • Google / Firebase — infrastructure cloud (Auth, Firestore, Hosting)
  • face-api.js / TensorFlow.js — modèles de reconnaissance faciale open-source
  • Telegram — canal de notifications instantanées
  • Navigateurs modernes — accès caméra via WebRTC
  • Hébergeur académique / entreprise cliente — déploiement & maintenance
  • GitHub — versioning & collaboration open-source
⚙️

Activités Clés

  • Développement & maintenance de l'app React
  • Entraînement & intégration des modèles de détection faciale
  • Gestion des données Firestore (employés, pointages)
  • Envoi de notifications Telegram en temps réel
  • Administration des accès (rôles Admin / Employé)
  • Évolutions fonctionnelles & correctifs sécurité
💎

Proposition de Valeur

Pointer sans badge, sans contact, sans friction.

  • ✅ Pointage 100 % automatique via reconnaissance du visage
  • ✅ Zéro carte magnétique, zéro code PIN à mémoriser
  • ✅ Alertes instantanées sur Telegram (entrée / sortie)
  • ✅ Dashboard temps-réel : présents, absents, retards
  • ✅ Sécurité renforcée — impossible d'usurper une présence
  • ✅ Déployable sur n'importe quel PC avec caméra — aucune infra propriétaire
  • ✅ Données centralisées & accessibles depuis n'importe où
  • ✅ Open-source & personnalisable selon les besoins
💬

Relations Clients

  • Interface self-service pour les employés (Punch page)
  • Dashboard admin dédié aux responsables RH
  • Notifications Telegram automatiques post-pointage
  • Support technique via GitHub Issues / email
  • Documentation & guide d'onboarding intégré
👥

Segments Clients

  • 🏢 PME & startups cherchant à digitaliser leurs RH
  • 🎓 Établissements scolaires / universités (suivi de présence étudiants)
  • 🏭 Usines & entrepôts (contrôle des entrées / sorties)
  • 🏥 Cliniques & établissements de santé (personnel soignant)
  • 🏛️ Administrations publiques (contrôle des fonctionnaires)
  • 👨‍💼 DRH & responsables administratifs
  • 🔬 Laboratoires & centres de recherche
🧱

Ressources Clés

  • Codebase React (Vite) + services modulaires
  • Firebase (Firestore DB, Authentication, Hosting)
  • Modèles face-api.js (SSD, landmark, descriptor)
  • Bot Telegram configuré & actif
  • Webcam standard (matériel client)
  • Expertise développement & IA embarquée
📡

Canaux

  • Application Web (navigateur) — accès direct
  • Firebase Hosting — déploiement cloud instantané
  • Telegram — notifications push temps-réel
  • GitHub — distribution open-source & démos
  • Réseaux sociaux / LinkedIn — acquisition B2B
  • Démonstrations en présentiel (salons RH, académique)
💸

Structure de Coûts

  • Firebase Spark (gratuit) → Blaze (pay-as-you-go) selon volume
  • Nom de domaine personnalisé (~10 €/an)
  • Coût de développement initial (temps humain)
  • Maintenance & mises à jour régulières
  • Serveur/hébergement si déploiement on-premise
  • Formation des utilisateurs & administrateurs
  • Coût matériel caméra (supporté par le client)
  • Éventuels frais API Telegram si volume élevé (actuellement gratuit)
💰

Sources de Revenus

  • 🎓 Projet académique (valorisation / note) — phase actuelle
  • 💼 Licence SaaS mensuelle par entreprise (ex: 49–299 €/mois)
  • 🏭 Déploiement on-premise personnalisé (prestation forfaitaire)
  • 🔧 Contrat de maintenance & support annuel
  • ⚙️ Intégrations sur mesure avec logiciels RH tiers (ERP, paie)
  • 📈 Vente de rapports & analytics avancés (fonctionnalité premium)
  • 🎓 Vente à des institutions académiques (licence campus)
  • 🚀 Levée de fonds / incubateur si passage en startup
